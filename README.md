# hashdump
Get hashes for recursive files (windows/unix)


    usage: hash.py [-h] [-md5] [-sha1] [-sha256] path

    positional arguments:
      path        full path

    optional arguments:
      -h, --help  show this help message and exit
      -md5        md5 hash
      -sha1       sha1 hash
      -sha256     sha256 hash
